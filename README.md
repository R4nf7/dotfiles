# dotfiles

## Install

```bash
git clone https://github.com/R4nf7/dotfiles.git ~/dotfiles
echo "source ~/dotfiles/zshrc" >> ~/.zshrc
echo "source ~/dotfiles/vimrc" >> ~/.vimrc
```
